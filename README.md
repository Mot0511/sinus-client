# Pipeup (прошлое название - Sinus) - простая социальная сеть
Это примитивная социальная сеть, в которой у каждого пользователя есть своя страничка, и пользователи могут публиковать посты и переписываться. Этот проект я создавал для практики навыков работы с FastApi (python фреймворк). Проект разделен на frontend и backend части, которые работают отдельно (frontend размещен на сервисе vercel.com, а backend на render.com), у них отдельные github репозитории. В первый раз сайт может долго грузиться (около минуты) из-за бесплатного сервера для backend части.

*Это репозиторий фронтенд части проекта, репозиторий бэкенда можно посмотреть [здесь](https://github.com/Mot0511/sinus-server).*

**Используемые технологии во фронтенде:**
- HTML/CSS/JS
- Typescript
- React
- NextJS

Ссылка на сайт - https://pipeup.vercel.app
